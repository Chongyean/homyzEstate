## Run Locally

Clone the project

```bash
  git clone https://github.com/Chongyean/homyzEstate.git
```

Go to the project directory

```bash
  cd homyzEstate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
